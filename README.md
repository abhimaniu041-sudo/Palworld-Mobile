# 🐾 Palworld Mobile (Zenith Edition)

A Premium Open-World Survival Craft (SOC) game built for mobile devices.

## 🚀 Core Features
- **4km² Dynamic Map:** Explore Hills, Deserts, Snow, and Waterfalls.
- **Monster System:** 50+ Unique Pals with different abilities and sizes.
- **Advanced Breeding:** Genetic inheritance system for powerful partners.
- **Automation:** Assign Pals to your base for mining, farming, and crafting.
- **Premium UI:** High-fidelity Red & Black gaming aesthetic.

## 📁 Project Structure
- `/src/core`: Game Engine & Physics
- `/src/entities`: Monster AI & Breeding
- `/src/systems`: Inventory, Catching & Building
- `/src/ui`: Premium Dashboard & Styles

## 🛠 Tech Stack
- JavaScript (ES6+)
- HTML5 Canvas / Mobile Web Optimization
- LocalStorage Data Persistence
